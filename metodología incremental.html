Metodología incremental

La metodología incremental es un modelo de desarrollo de software en el que el sistema se construye y entrega por partes llamadas incrementos. Cada incremento agrega nuevas funciones hasta completar el producto final.

Características

-Desarrollo por módulos o partes.
-Entregas parciales del sistema.
-Permite detectar errores rápidamente.
-El cliente puede probar avances constantemente.

¿Cómo funciona?

El proyecto se divide en pequeñas etapas.
En cada una se:

1. Analizan requisitos.
2. Diseña una parte del sistema.
3. Se programa.
4. Se prueba.
5. Se entrega una versión funcional.

Luego se continúa con otro incremento que añade más características.
